# PayController

## Tecnologias

- Kotlin
- XML (layouts)
- Room (base de datos local)
- LiveData y ViewModel (arquitectura MVVM)
- RecyclerView y CardView
- MPAndroidChart (grafico de dona)
- Material Design Components
